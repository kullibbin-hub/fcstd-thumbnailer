# fcstd-thumbnailer
Скрипт реализует создание превью для файлов FreeCAD формата .FCstd для Nautilus. Проверен в Altlinux Workstation 11.1, возможно будет работать в Ubuntu.
Для Altlinux потребуется включить sudo. В терминале

su -

пароль

control sudowheel enabled

Скрипт создает два файла /usr/local/bin/fcstd-thumbnailer и /usr/share/thumbnailers/fcstd.thumbnailer. Смысл - распаковать файл проекта 
.FCstd, вытащить из него превью в формате .png и передать в nautilus для отображения.

Для работы предпросмотра должен быть установлен FreeCAD, любым способом, в том числе и из Flatpak, и файлы .FCstd должны открываться по умолчанию в нем, что обычно и происходит.
В самом FreeCAD должна быть включена опция сохранения миниатюры в файл проекта (Настройки - Документ - Сохранять миниатюру в файл проекта).
Размер - 256.

Использование.

По зеленой кнопке "<>Code" вверху страницы выбрать Download zip, скачается архив. Распаковать в домашний каталог (можно оставить в папке "Загрузки"), в распакованной папке в свойствах скрипта "fcstd-thumbnailer-install.sh" по правому клику включить "Исполняемый как приложение". Правый клик по скрипту - копировать, открыть терминал, правый клик в терминал - вставить. Enter. Желательно чтоб скрипт не находился в папке с пробелами, например, "Рабочий стол" или "мой софт". Лучше переместить его в корень домашнего каталога, или оставить в папке "Загрузки", как уже писал. Также, можно запустить скрипт правым кликом по нему - "запустить как приложение".

После отработки скрипта, для гарантии, выйти из сессии и зайти снова, или перезагрузить компьютер. Теперь nautilus будет показывать превью
сохраненных проектов FreeCAD.

4.03.2026. Update: Исправлена ошибка, при которой для FreeCAD, установленного из репозитория, превью могли не создаваться.

Данный скрипт можно свободно распространять, модифицировать и использовать без каких-либо ограничений, на свой страх и риск.
