# fcstd-thumbnailer
Скрипт реализует создание превью для файлов FreeCAD формата .FCstd для Nautilus. Проверен в Altlinux Workstation 11.1, возможно будет работать в Ubuntu.

Для работы предпросмотра должен быть установлен FreeCAD, и файлы .FCstd должны открываться по умолчанию в нем, что обычно и происходит.
В самом FreeCAD должна быть включена опция сохранения миниатюры в файл проекта (Настройки - Документ - Сохранять миниатюру в файл проекта).
Размер - 256.

Использование.

По зеленой кнопке "<>Code" вверху страницы выбрать Download zip, скачается архив. Распаковать в домашний каталог (можно оставить в папке "Загрузки"), в распакованной папке в свойствах скрипта "fcstd-thumbnailer.sh" по правому клику включить "Исполняемый как приложение". Правый клик по скрипту - копировать, открыть терминал, правый клик в терминал - вставить. Enter. Желательно чтоб скрипт не находился в папке с пробелами, например, "Рабочий стол" или "мой софт". Лучше переместить его в корень домашнего каталога, как уже писал. Также, можно запустить скрипт правым кликом по нему - "запустить как приложение".

После отработки скрипта выйти из сессии и зайти снова, или перезагрузить компьютер.
